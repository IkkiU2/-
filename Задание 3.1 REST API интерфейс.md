
HTTP метод: POST
URL: /api/users/register
Content-Type: application/json
Входные параметры (Request Body)

Параметр	Тип	Обязательность	Ограничения / Описание
firstName	string	Да	1-50 символов
lastName	string	Да	1-50 символов
userName	string	Да	3-30 символов, только латинские буквы, цифры и подчеркивание
password	string	Да	Минимум 8 символов, должен содержать: хотя бы одну цифру, одну заглавную букву, одну строчную букву, один специальный символ
captchaToken	string	Да	Токен reCAPTCHA от Google
Успешный ответ (HTTP 201 Created)

json
{
  "success": true,
  "message": "User Register Successfully.",
  "data": {
    "userId": 12345,
    "firstName": "ivan",
    "lastName": "ivanov",
    "userName": "ivan",
    "createdAt": "2026-02-05T00:35:43Z"
  }
}
Ответы с ошибками

400 Bad Request (Ошибки валидации)

json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Please verify reCaptcha to register!",
    "details": {
      "field": "captchaToken",
      "requirement": "Captcha verification is required"
    }
  }
}
409 Conflict (Пользователь уже существует)

json
{
  "success": false,
  "error": {
    "code": "USER_EXISTS",
    "message": "User exists!",
    "details": {
      "field": "userName",
      "value": "ivan"
    }
  }
}
422 Unprocessable Entity (Ошибка пароля)

json
{
  "success": false,
  "error": {
    "code": "PASSWORD_VALIDATION",
    "message": "Password validation failed",
    "details": {
      "requirement": "Passwords must have at least one non alphanumeric character, one digit ('0'-'9'), one uppercase ('A'-'Z'), one lowercase ('a'-'z'), one special character and Password must be eight characters or longer."
    }
  }
}
Коды ответов и ошибки

Код HTTP	Тип ошибки	Сообщение	Когда возникает
201	Успех	"User Register Successfully."	Регистрация прошла успешно
400	Клиентская	"Please verify reCaptcha to register!"	Не пройдена проверка капчи
400	Клиентская	"Invalid request parameters"	Неправильный формат JSON
409	Клиентская	"User exists!"	Пользователь с таким userName уже существует
422	Клиентская	"Password validation failed"	Пароль не соответствует требованиям
422	Клиентская	"Validation failed for field: [fieldName]"	Ошибка валидации других полей
500	Серверная	"Internal server error"	Ошибка на сервере (БД и т.д.)
