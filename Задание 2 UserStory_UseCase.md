2.1. User Story

Как продавец на маркетплейсе
Я хочу самостоятельно публиковать товары через личный кабинет
Чтобы быстро добавлять новые товары на витрину без помощи поддержки и увеличивать продажи.

Критерии завершённости:

Продавец видит кнопку «Добавить товар» в личном кабинете
При нажатии открывается форма с полями: название, категория, цена, фото
Система проверяет правильность заполнения и показывает ошибки
После публикации товар появляется на витрине или отправляется на модерацию
Продавец видит статус товара (Черновик/На модерации/Опубликован)
2.2. Use Case

Название: Добавление нового товара на маркетплейс

Участники: Продавец, Система маркетплейса

Предусловия:

Продавец авторизован в личном кабинете
У продавца есть активный магазин на платформе
Основной сценарий:

Шаг	Действие продавца	Ответ системы
1	Нажимает «Добавить товар»	Открывает форму создания товара
2	Заполняет поля:
• Название товара (обязательно)
• Категория (обязательно)
• Цена (обязательно, >0)
• Количество (обязательно)
• Фотографии (1-10 штук)	Показывает подсказки для каждого поля
3	Нажимает «Опубликовать»	Проверяет данные на ошибки
4		Если ошибок нет → сохраняет товар
5		Если товар из простой категории → публикует сразу
6		Если товар из сложной категории → отправляет на модерацию
7		Показывает сообщение об успехе
Результат: Товар добавлен в систему с статусом «Опубликован» или «На модерации»
